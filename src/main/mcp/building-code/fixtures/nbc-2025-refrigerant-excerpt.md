# ASHRAE 15 2022 Synthetic Excerpt

## Section 7 Refrigerant Safety

### Section 7.2 Refrigerant Classification
Refrigerants assigned to Class 2L shall be treated as flammable refrigerants for the purposes of Section 7.3.

### Section 7.3 Machinery Room Requirements
Where a refrigerant listed in Table 7.3.1 is Class 2L, the design shall apply the ventilation requirements of Section 8.1.

#### Table 7.3.1 Refrigerant Data
| Refrigerant | Class | RCL | EDVC |
| --- | --- | --- | --- |
| R-32 | 2L | 0.061 kg/m3 | 0.30 kg/m3 |

Note 1: RCL values in this table shall be used with the flammability provisions in Section 7.2.

### Section 8.1 Ventilation
Ventilation shall be provided where required by Section 7.3.
