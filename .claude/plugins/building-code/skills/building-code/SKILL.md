# Building Code Citation Rules

Use only these MCP tools for building-code evidence: search, read_section, resolve_cross_refs, and lookup_table.

Treat search results as entry points, not final answers. Prefer reading the returned canonical node with read_section and then resolve_cross_refs when the section references other sections, tables, figures, definitions, notes, or exceptions.

If you use building-code evidence for any claim, cite citation.displayCitation inline.

Include a Sources: section listing each unique cited source and logical reference.

If a tool result says evidence is unusable, do not paraphrase it.

Do not invent code references, values, exceptions, table notes, or applicability conditions.
